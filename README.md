# Пет проект системного аналитика "Оформление банковского вклада"

<h3>О проекте</h3>
Этот проект демонстрирует полный цикл системного анализа при разработке распределенной банковской системы. 

<h3>Архитектура</h3>

В проекте участвуют 4 внутренних микросервиса и 2 внешние системы:
- **msa_deposit** (Вклады) — оркестратор процесса.
- **msa_client** (Клиенты), **msa_contract** (Договоры), **msa_account** (Счета).
- Интеграция с **ФНС** (SOAP/XML) и внутренней **Бухгалтерией**.

<h3>Ключевые артефакты</h3>

- BPMN-схема оформления банковоского вклада
- Use Cases и User Story
- UML диграмма последовательности — логика вызовов между сервисами.
- Макеты в Figma — интерфейс пользователя.
- JSON- и XSD-схемы и примеры их использования
- OpenAPI-спецификация — контракт REST API.
- ТЗ для разработчиков

<h3>Стек технологий</h3>

- Моделирование: BPMN, UML, PlantUML
- Документирование API: OpenAPI 3.0 (Swagger), YAML
- Данные: JSON Schema, XML/XSD
- Прототипирование: Figma
- Инструменты: Swagger Editor
